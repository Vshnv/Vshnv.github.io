JWeb
A Java based API to allow creation of static websites using Java replacing HTML/CSS/JS. JWeb also adds use of 'Components' to reduce the redundency in Code.
Description
JWeb converts Java code into html,css and js to allow creation of clean and easy to create websites. The Object model present allows you to create components which can be reused thus greatly reducing redundency. It also allows the use of external Modules which allow further expansion on the project. JWeb can also be used as a Server Software in which case it can be used to make Session based changes to the pages.
